# OpenAIM Mac 客户端 UI 设计说明

## 设计概览

本文档描述 OpenAIM Mac 客户端的 UI 设计规范。

## 配色方案

### 主色调
| 名称 | 色值 | 用途 |
|------|------|------|
| 主蓝色 | `#3B82F6` | 按钮、头像、强调元素 |
| 浅蓝色 | `#EFF6FF` | 选中项背景 |
| 在线绿 | `#22C55E` | 在线状态、成功提示 |

### 背景色
| 名称 | 色值 | 用途 |
|------|------|------|
| 背景灰 | `#F8FAFC` | 页面背景 |
| 次级背景 | `#F1F5F9` | 输入框背景、次级背景 |
| 卡片白 | `#FFFFFF` | 卡片、侧边栏 |

### 文字色
| 名称 | 色值 | 用途 |
|------|------|------|
| 标题 | `#1E293B` | 标题、重要文字 |
| 正文 | `#334155` | 正文内容 |
| 次级 | `#64748B` | 描述、次要信息 |
| 辅助 | `#94A3B8` | 占位符、辅助文字 |

### 边框色
| 名称 | 色值 | 用途 |
|------|------|------|
| 边框灰 | `#E2E8F0` | 边框、分割线 |

## 界面设计

### 1. 登录界面 (LoginView)
- 左侧：蓝色品牌区 (`#3B82F6`)
  - Logo 图标
  - 品牌名称 "OpenAIM"
  - 标语 "AI-Powered Communication Platform"
- 右侧：白色表单区
  - 标题 "Welcome Back"
  - 邮箱输入框
  - 密码输入框
  - 登录按钮 (蓝色)
  - 忘记密码链接
  - 分割线
  - Google 登录按钮
  - 注册链接

### 2. 注册界面 (RegisterView)
- 左侧：蓝色品牌区 (同登录)
- 右侧：白色表单区
  - 标题 "Create Account"
  - 姓名输入框
  - 邮箱输入框
  - 密码输入框
  - 确认密码输入框
  - 创建账户按钮
  - 登录链接

### 3. 主界面 (MainView)
- 左侧边栏 (280px)
  - 用户头像 + 名称
  - 新建聊天/设置按钮
  - 搜索框
  - 会话列表
    - 头像 (圆形)
    - 名称
    - 消息预览
    - 时间
    - 未读数徽章
- 右侧聊天区域
  - 头部：对方信息 + 操作按钮
  - 消息列表
    - AI 消息：左对齐，白色气泡
    - 用户消息：右对齐，蓝色气泡
  - 输入框：附件 + 输入 + 发送

### 4. Agent 列表界面 (AgentListView)
- 左侧边栏
  - 标题 "My Agents"
  - 创建按钮
  - 搜索框
  - Agent 列表
    - 头像 (圆角方形)
    - 名称
    - 描述
    - 状态指示器
- 右侧详情区
  - Agent 信息
  - Access Token 卡片
  - Skills 标签
  - 组织列表

### 5. 创建 Agent 界面 (CreateAgentView)
- 标题 "Create New Agent"
- 名称输入框
- 描述输入框
- Skills 多选
- 取消/创建按钮

### 6. 组织管理界面 (OrganizationView)
- 左侧边栏
  - 标题 "Organizations"
  - 创建按钮
  - 组织列表
- 右侧详情区
  - 组织信息
  - 邀请按钮
  - 成员列表

## 组件规范

### 按钮
- 主按钮：蓝色背景 `#3B82F6`，白色文字
- 次级按钮：白色背景，灰色边框
- 圆角：8-10px
- 高度：44px (大) / 36px (中) / 32px (小)

### 输入框
- 高度：44px
- 圆角：10px
- 边框：1px `#E2E8F0`
- 背景色：`#FFFFFF`

### 卡片
- 背景色：`#FFFFFF`
- 圆角：12px
- 边框：1px `#E2E8F0`
- 内边距：20px

### 头像
- 圆形：用于用户、会话
- 圆角方形：用于 Agent、组织
- 尺寸：80px (大) / 48px (中) / 40px (小) / 36px (最小)

### 徽章
- 圆角：6px 或 100px (胶囊形)
- 内边距：4px 10px

## 间距规范

| 名称 | 值 | 用途 |
|------|------|------|
| xs | 4px | 紧凑间距 |
| sm | 8px | 小间距 |
| md | 12px | 中间距 |
| lg | 16px | 大间距 |
| xl | 24px | 区块间距 |
| xxl | 32px | 页面边距 |

## 字体规范

- 字体：SF Pro (系统字体)
- 标题：24-28px, Bold
- 副标题：18px, Semibold
- 正文：14-15px, Regular
- 辅助：12-13px, Regular
- 代码：JetBrains Mono
